Privacy Policy

1. Introduction

This Privacy Policy outlines how CommunityCTRL collects, uses, and protects your personal data when using our services.

2. Data Collection and Use

We collect user data for providing and enhancing our services. This includes user registration, access control, and security purposes.

3. User Information

User data includes personal information such as name, email, phone number, and unit information. This data is used for managing access, communications, and administrative purposes.

4. Visitor Management Data

Visitor information collected through our system includes name, contact details, and access permissions for security monitoring and access control within the community.

5. Security Measures

We implement security practices to protect user data from unauthorized access, including encrypted data storage and restricted access to sensitive information.

6. License Plate Recognition Data

The system collects and securely stores license plate data for community access control, accessible only to authorized personnel. This data is retained for security purposes and is periodically reviewed, ensuring compliance with data protection standards and shared only when legally required.

7. Data Retention

User data is retained only for as long as necessary to fulfill our services. Data may be reviewed periodically to comply with data protection regulations.

8. Contact Us

If you have any questions about our Privacy Policy, please contact us at communityctrl.service@gmail.com.
